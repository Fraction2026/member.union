## التاريخ
19 يونيو 2026

## المشكلة الأصلية
كان النظام يمنع رقم العضوية المكرر في **جميع اللجان** بدلاً من منعه في **نفس اللجنة فقط**.

## القواعد المطلوبة
1. **منع التكرار العالمي**: إذا كان (الاسم + الرقم القومي + تاريخ الميلاد) متطابقين في **أي لجنة**
2. **منع التكرار المحلي**: إذا كان رقم العضوية متطابق في **نفس اللجنة فقط**

## التعديلات المنفذة

### الملف: `/app/backend/server.py`
#### الدالة: `_find_all_member_duplicates` (السطر 2542)

**قبل التعديل:**
```python
# البحث برقم العضوية في كل اللجان
if membership_number:
    or_clauses.append({"membership_number": membership_number})
```

**بعد التعديل:**
```python
# البحث برقم العضوية في نفس اللجنة فقط
if membership_number and union_committee and department_id:
    or_clauses.append({
        "membership_number": membership_number,
        "union_committee": union_committee,
        "department_id": department_id
    })
```
